## PORT

### 만약 하나의 컴퓨터에서 게임과 화상통화, 웹 브라우저 요청 등이 한 번에 이루어지면 컴퓨터는 어떻게 한 번에 여러 개를 연결할까? <br>
- 패킷이 날아왔을 때 게임에서 필요한 것인지, 화상통화에서 필요한 것인지 IP만으로는 알 수가 없다.
- 그래서 TCP 세그먼트에 출발지 PORT, 목적지 PORT를 기재하여 구분한다.
- 즉, PORT는 같은 IP 내에서 프로세스를 구분하기 위해 사용한다. IP는 아파트 건물, PORT는 각 호수라고 보면 된다.

### 잘 알려진 포트
- FTP - 20, 21
- TELNET - 23
- HTTP - 80
- HTTPS - 443

해당 포트는 피해서 사용하는 것이 좋다.
