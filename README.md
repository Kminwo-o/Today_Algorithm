- 그리디 : 이미 이전에 풀어서 그리디인걸 알고 있었다. 모든 조합을 찾으며 비교하기에는 T * N (최대 1만)이기 때문에 굉장한 시간이 걸린다. 따라서 해당 문제는 그리디로 풀어야한다.
