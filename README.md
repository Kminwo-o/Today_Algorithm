### 로봇 시뮬레이션
다른 시뮬레이션과 큰 차이는 없었지만, 문제를 제대로 읽지않는 일이 빈번한 나에게는 매우 어려운 문제였다. <br>
다른 건 없었다. 단지 좌표의 시작점이 왼쪽 아래이기 때문에 상하를 반대로 지정해주고 x, y 좌표 이동을 2차원 배열과 다르게 해야했다. <br>
괜히 시간을 몹시 잡아 먹혔다.
